# Online Retail Sales SQL Project

## About the Project
This project is a simple SQL-based Online Retail Sales system.
It is created to understand how real-world sales data is stored and analyzed
using relational databases.

The project includes customers, products, and orders data
and focuses on writing SQL queries for business analysis.

---

## What I Did in This Project
- Created tables for customers, products, and orders
- Inserted sample sales data
- Used joins to combine data from multiple tables
- Wrote queries to find top-selling products
- Calculated monthly revenue
- Identified inactive and high-value customers
